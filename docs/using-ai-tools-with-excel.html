# Using AI tools with Excel

Many AI tools are embedded into Excel. We have already used some of these features in previous chapters. When you create a new variable in a new column using a formula, Excel will use it as an example to write the formula in all subsequent rows in the same column and automatically update as the data changes ("Formula by Example"). Similarly, when creating a chart, Excel proposes a selection of charts that are the most likely to be relevant in "Recommended charts".

In this chapter, we will present how Large Language Models (LLM) can be used to 1) write VBA code to perform tasks in Excel and 2) perform tasks directly in Excel using Microsoft Copilot.

The key messages are:

1. There is a code behind every task performed in Excel and AI tools work by transforming prompts into code.
2. Whether you are asking AI tools to write code or to perform a task directly, the main difficulty is to write a prompt that will get the desired task done.

This chapter was written in March 2024. By the time you read this, AI tools might have become more efficient at performing the tasks asked of them here.

## What are Large Language Model (LLM)?

Go to [Microsoft Copilot](https://copilot.microsoft.com/) or [ChatGPT](https://chat.openai.com/) and write the following prompts: 

`Explain what is Microsoft Copilot / ChatGPT / Large language models`

`Explain what is Excel VBA`

## What is Excel VBA?

VBA stands for **Visual Basic for Applications**. It is a programming tool that can be used to perform operations in Excel (and other office programs). Most operations performed in Excel can be programmed in VBA instead of being performed manually as we have done in the previous chapters. It is mostly useful when you have to do repetitive tasks, in which case it may be convenient to automate them using code, which can be saved and reused at a later stage. 

An important drawback of using Excel as we have done so far is that is that the tasks we have performed are not easily reproducible. You can save the results for later information but may not remember what tasks were performed to arrive at these results. 

However, it does not mean that coding in Excel VBA is the appropriate method to use at all times! It will soon be possible to ask Microsoft Copilot to perform simple tasks in Excel.
